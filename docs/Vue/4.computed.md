# computet 计算属性

日常开发中，当组件中的某一个数据需要依赖于其他数据的改变进行改变时，计算属性便是非常好的选择,我们看一个实例

```vue
<template>
  <div>
    <div>{{ isTrue }}</div>
  </div>
</template>
<script lang="ts" setup>
import { computed, ref } from 'vue';
const arr = ref([1, 2, 3]);

const isTrue = computed(() => {
  return arr.length > 0 ? 'true' : 'false';
});
</script>
<style lang="less" scoped></style>
```
